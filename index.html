Who?

My name is Andres Francisco. I have a bachelor degree in Engineering in Innovation and Design in which I studied all about how to create a product, service, applications and a lot more stuff. My degree goes all the way from the conception of ideas (conceptual phase) until the selection, prototyping, development and posterior implementation.

I also have a specialty degree in graphic simulation where I learned how to create environments, videogames and simulations while using software such as C++ with OpenGL, Unity with C# scripting and Unreal Engine while working with blueprints. I know my way around Virtual Reality development using unity and Oculus Quest.

A couple years ago coding changed my mind and made me want to know more about it, specifically Frontend Development. I started studying more, structuring languages such as HTML & CSS. After that I started studying JavaScript, Github and frameworks like React.
